# Changelog

## [1.0.0] — 2025-06-12

### 🎉 首次正式发布

VFX Audio Manager — 现代化桌面音效管理软件，为视频剪辑创作者与内容创作者打造。

### 核心功能

**音效导入与管理**
- 支持导入本地文件夹，自动递归扫描所有子目录
- 拖拽文件夹到应用窗口即可导入
- 支持 11 种音频格式：MP3、WAV、FLAC、OGG、M4A、AAC、WMA、AIFF(AIF)、OPUS、WEBM、MKA
- 每种格式有独立颜色标签，一目了然
- 导入时自动检测音频文件时长（并行批量处理）
- 启动时自动恢复上次的所有分类和文件列表
- 分类可右键重命名或删除，不影响磁盘文件
- 多级子目录树形浏览，每层可独立展开/折叠，支持无限层级
- 文件重命名（仅软件内，不修改磁盘文件）
- 从列表移除文件（仅软件内，不删除磁盘文件），持久化忽略列表

**音频播放**
- 播放/暂停/停止控制
- 可拖拽调整宽度的右侧播放面板
- 进度条拖动快进快退
- 等宽数字时间戳
- 音量滑块调节（0–100%），静音恢复之前音量
- 单例播放：同一时间只播放一首

**播放队列与循环**
- 右键添加到播放队列，播完自动下一首
- 三种循环模式：关闭 / 单曲循环 / 全部循环

**变速播放**
- 六档变速：0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x

**睡眠定时**
- 15 / 30 / 60 分钟定时自动暂停播放

**最近播放**
- 自动记录最近播放的 20 首音效
- 按时间倒序回溯
- 自动切换曲目也会记录

**搜索与排序**
- 实时搜索：按文件名和格式筛选，带清除按钮
- 四种排序：名称、文件大小、时长、格式，支持升降序
- 收藏文件自动排前
- 重命名文件按自定义名称排序

**收藏夹系统**
- 星标按钮一键收藏/取消收藏
- 自定义收藏夹：创建、重命名、删除
- 文件可同时属于多个收藏夹

**批量操作**
- 多选文件，批量收藏、添加到收藏夹、从列表移除
- 全选/取消全选当前可见文件

**键盘快捷键（全部可自定义绑定）**

| 快捷键 | 功能 |
|--------|------|
| Space | 播放 / 暂停 |
| Escape | 取消选择 / 停止播放 |
| Delete | 删除已选文件 |
| Ctrl+A | 全选当前列表 |
| Ctrl+F | 聚焦搜索框 |
| Ctrl+Q | 加入播放队列 |
| ← → | 快退 / 快进 5 秒 |
| ↑ ↓ | 音量增减 5% |
| M | 静音 / 取消静音 |

**右键菜单**
- 播放、添加到播放队列、添加到收藏夹、在资源管理器打开、复制路径、重命名、从列表移除

**设置页面**
- 默认音量、深色/浅色主题切换、自定义快捷键绑定、数据清除

**界面设计**
- 深色/浅色双主题，CSS 变量驱动即时切换
- 可拖拽分隔条调整三栏宽度，偏好持久保存
- 响应式音频卡片网格布局
- 空状态引导提示

**技术架构**
- Electron 42 + React 18 + TypeScript 5.6
- Vite 5 + esbuild 双构建
- TailwindCSS 3 + Zustand 4
- music-metadata 音频解析
- contextIsolation + IPC 安全通信
