# Changelog

## [1.0.0] - 2026-07-04

### 已添加

#### 核心内容
- 基于 Electron + React + TypeScript 的桌面音效管理应用
- 文件夹导入功能，支持递归扫描所有子目录
- 支持 12 种常见音频格式识别与元数据解析（MP3/WAV/FLAC/OGG/M4A/AAC/WMA/AIFF/OPUS/WEBM/MKA 等）
- 拖放文件夹到窗口直接导入

#### 浏览与搜索
- 多维度浏览：全部音效 / 收藏 / 最近播放 / 分类 / 子目录 / 自定义合集
- 实时关键词搜索过滤
- 四种排序方式：名称、文件大小、时长、格式，支持升降序

#### 收藏与合集
- 一键收藏/取消收藏音效文件
- 自定义合集（User Collections）：创建、重命名、删除合集
- 向合集中添加/移除音效文件

#### 音频播放器
- 内建音频播放器，封装 HTML5 Audio API
- 播放/暂停、停止、进度条拖动
- 音量控制（0～100%）与静音切换
- 三种循环模式：关闭 / 单曲循环 / 列表循环
- 变速播放：0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x
- 播放队列（添加/移除/清空/播放下一首）
- 睡眠定时器（5/10/15/30/60 分钟）

#### 批量操作
- 多选模式：Ctrl/Shift + 点击选择
- 全选/取消全选
- 批量收藏/取消收藏
- 批量从列表中移除

#### 键盘快捷键
- 完整的快捷键系统，覆盖播放控制、导航、音量、搜索、队列等 11 项操作
- 支持在设置中自定义所有快捷键
- 录制模式：点击快捷键按钮后按下新键即可替换

#### 主题
- 5 种内置主题：暗黑、明亮、暖色、森林、海洋
- 通过 Tailwind CSS 自定义颜色系统实现

#### 数据持久化
- 通过 Electron IPC + 本地 JSON 文件持久化存储
- 持久化的数据包括：分类列表、收藏列表、自定义合集、合集文件关联、应用设置、已忽略路径
- 支持旧版数据自动迁移（importFolders → categories）
- 重置所有数据功能

#### UI/UX
- 可拖拽调整侧边栏宽度（160px ~ 480px）
- 可拖拽调整播放器面板宽度（200px ~ 500px）
- 文件右键菜单：在资源管理器中定位、复制完整路径
- 自定义文件显示名称（不影响原文件名）
- 最近播放列表（最多 20 条）
- 导入进度指示器（扫描中/解析中状态提示）

#### 安全
- contextIsolation + preload 脚本，禁用 nodeIntegration
- 生产环境内容安全策略（CSP）

#### 开发体验
- Vite 5 开发服务器 + HMR
- TypeScript 严格类型检查
- esbuild 构建 Electron 主进程
- electron-builder NSIS 安装程序打包
- 并发启动 Vite + Electron 的开发脚本
